macOS 14+ Swift 5.9 No dependencies v1.3.0

Everything you need, one click away

Four purpose-built tabs live in your menu bar. No Dock icon, always accessible.

🗂

Asset Vault

Drag images and text from anywhere. Organize into folders. Drag back out to Keynote, Pages, or any app. Persistent JSON storage.

⇄

Smart Transform

Paste content — Clip auto-detects Markdown, HTML, CSV, TSV, or plain text and offers the right conversion. Side-by-side split view for comparison.

𝐀

Text Formatter

Unicode styling for social media: bold, italic, monospace, script, small caps. Convert full Markdown docs to Unicode-styled plain text.

📄

Snippets

Store reusable text snippets with titles. One click to copy. Built-in templates for common patterns.

💾

Export / Import

Back up your entire vault as JSON. Restore on any Mac. Works from the menu bar or CLI. Schedule automated backups with cron.

🔄

Auto-Update

Clip checks GitHub Releases on launch. Get notified when a new version is available and download with one click.

Getting Started

Up and running in under a minute.

Download & install

Download Clip-1.3.0.dmg from the latest release. Open the DMG, drag Clip.app to /Applications.

Bypass Gatekeeper

The app isn't notarized yet. Run this once in Terminal:

xattr -rd com.apple.quarantine /Applications/Clip.app

Launch

Double-click Clip.app. The paperclip icon appears in your menu bar. Left-click to open the panel, right-click for the menu.

Install the CLI (optional)

Double-click "Install CLI.command" in the DMG to install the clip terminal tool to /usr/local/bin.

Quick access

Use Cmd+Shift+C from any app, or shake your mouse vigorously to reveal the panel instantly.

clip CLI

The same transformers available in the app, usable from your terminal.

Transform

echo "# Hello **world**" | clip md2html echo "<h1>Hello</h1>" | clip html2md echo "**bold**" | clip md2social # Clipboard mode — no pipes needed clip md2html --clipboard

Backup & Restore

# Export vault clip export > ~/backup.json clip export --file ~/Dropbox/clip.json # Import vault clip import < ~/backup.json clip import --file ~/Dropbox/clip.json # Automated daily backup 0 2 * * * clip export --file ~/backup.json

Build from source

Requires Xcode Command Line Tools or full Xcode.

git clone https://github.com/blacklogos/markutils.git cd markutils swift run # build & launch the app bash scripts/install_cli.sh # install CLI to /usr/local/bin bash scripts/build_dmg.sh # build distributable DMG
