Clip

Your content toolkit, in the menu bar.

A macOS menu bar app + CLI that manages assets, transforms Markdown, and formats text for social media — with zero dependencies.

Download for Mac GitHub
clip 
# Transform Markdown to HTML
$ echo "# Hello **world**" | clip md2html
<h1>Hello <strong>world</strong></h1>

# Format for social media
$ echo "**bold** and _italic_" | clip md2social
𝐛𝐨𝐥𝐝 and 𝒊𝒕𝒎𝒐𝒊𝒌

# Back up your vault
$ clip export --file ~/Dropbox/clip-backup.json
Exported 42 assets to ~/Dropbox/clip-backup.json
🗂

Asset Vault

Drag images and text in, organise into folders, drag back out to any app. Persists across launches.

⇄

Smart Transform

Auto-detects Markdown, HTML, CSV, or TSV and offers the right conversion. Split view for side-by-side comparison.

𝐀

Text Formatter

Unicode styling for social media: bold, italic, monospace, script, small caps. Full Markdown-to-Unicode conversion.

Get started

# 1. Download & install
$ open https://github.com/blacklogos/markutils/releases/latest

# 2. Remove quarantine (app is not notarized)
$ xattr -rd com.apple.quarantine /Applications/Clip.app

# 3. Launch — look for the paperclip in your menu bar
$ open /Applications/Clip.app

# Quick access: Cmd+Shift+C or shake your mouse

Or build from source: git clone && swift run
